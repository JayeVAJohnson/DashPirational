# Philosophy

Built to run anywhere.

Single file. No dependencies. Offline by design, online affirming.
No frameworks required. No backend. No install.

Digital simplicity with access in mind — because the best tool
is the one that works on the rig in front of you.
Not the machine you wish you had...the one you have access to at the moment.

This shapes every decision: what gets included, what gets left out,
how the code is structured, what the output weighs.
If it can't run on modest hardware with spotty internet,
it isn't finished yet.

Wealth and resources are lovely, too--this work doesn't oppose them,
It just doesn't require them.
