Database Settings

Right now, this dashboard is using sample data only. Nothing is being saved permanently — everything resets when you refresh.

The nice part is that this dashboard is designed to grow with you. Whenever you're ready, you can plug in a real database.

You can use:
• A simple JSON file stored on your computer
• A SQLite file (a tiny real database)
• LocalStorage (built into your browser)
• Or even a cloud database later on

This dashboard is flexible on purpose. Swap in a different data source whenever you want.
