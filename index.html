DashPirational

DashPirational is a lightweight personal dashboard designed to serve as a daily cockpit, thinking space, and life‑OS foundation. It is built entirely with HTML, CSS, and JavaScript, with no backend dependencies.

About the Project

DashPirational includes:

  • Today page for quick capture and Kanban, now with drag‑and‑drop polish and saved state
  • Search page with local and web search, including in‑page match highlighting
  • Pages for Tasks, Routines, Projects, Content, People, and Library
  • Study Panel for focused learning
  • Floating theme switcher (Light, Dark, Lavender) with one‑click previews in Settings
  • Local‑storage persistence — your captures, tasks, routines, and board survive a refresh
  • Responsive layout with mobile breakpoints
