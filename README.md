DashPirational is a lightweight, frontend-only personal dashboard designed to serve as a daily cockpit, thinking space, and life-OS foundation. It runs entirely in the browser, requires no backend, and is easy to customize. - Jaye Johnson

## Features

- Today page with quick capture and embedded Kanban
- Search page with local dashboard search and Google Web Search
- Tasks, Routines, Projects, Content, People, Library pages
- Study Panel for active learning
- Floating theme switcher (Light, Dark, Lavender)
- Google Sheets Kanban embed
- Responsive layout
- Zero dependencies (pure HTML, CSS, JavaScript)

## Starter Foundation

This project is a starter foundation for a personal dashboard.

Right now, because it is purely frontend, anything typed into the text areas will disappear when the browser is refreshed. Future updates will introduce localStorage support, JSON export, and additional persistence features.

Three built-in themes are available:
